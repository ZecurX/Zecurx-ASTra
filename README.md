# Zecurx CodeGuard

Zecurx ASTra is a Static Application Security Testing (SAST) tool
that analyzes JavaScript and TypeScript source code without execution
to identify security vulnerabilities and bugs.

## Features

- Static code analysis (AST-based)
- Security vulnerability detection
- CLI and VS Code integration (WIP)
